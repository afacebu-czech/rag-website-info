## ðŸ”§ Troubleshooting

### "Can't paste image"
- Make sure you've **copied the image** to clipboard first
- **Click inside the upload box** before pasting
- Try using **drag and drop** instead
- Check that your browser supports clipboard paste (most modern browsers do)

### "Image not processing"
- Ensure image is in supported format (PNG, JPG, JPEG, GIF, BMP)
- Check that image is clear and text is readable
- Try a different image or format
- Verify OCR dependencies are installed: `pip install easyocr pillow`

### "Text not extracted correctly"
- Use a **higher resolution** image
- Ensure **good contrast** between text and background
- Try **cropping** to focus on the text area
- Check that text is **not rotated** or skewed
